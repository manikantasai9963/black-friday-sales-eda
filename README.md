# Black Friday Sales Analysis

This project performs Exploratory Data Analysis (EDA) on a Black Friday retail dataset to understand customer purchasing behavior.

## Tech Stack
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

## Analysis Performed
- Data cleaning and preprocessing
- Customer purchase patterns by age group
- Sales distribution across product categories
- Purchase trends by customer type
- State-wise sales insights

## Dataset
Black Friday Impact Dataset
